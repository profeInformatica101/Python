1. Administración de usuarios y permisos en Linux

¿Qué comandos usarías para realizar las siguientes tareas?

  1. Crear tres usuarios: usuario1, usuario2 y usuario3.
  2. Crear dos grupos: grupo1 y grupo2.
  3. Agregar:
    • usuario1 y usuario2 al grupo grupo1.
    • usuario3 al grupo grupo2.
  4. Configurar permisos en las carpetas:
    • /proyecto1: acceso total (lectura, escritura y ejecución) solo para los miembros de grupo1.
    • /proyecto2:
      • Lectura y escritura para los miembros de grupo2.
      • Solo lectura para los miembros de grupo1.
    • /compartido: acceso total para todos los usuarios.
    • /privado: acceso exclusivo para usuario1.
